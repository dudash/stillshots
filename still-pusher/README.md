still-pusher
==============================================================================
a simple node.js REST client app to watch a directory of images files and push
new image contents to a REST based server


Author(s)
-------------------------------------------------------------------------------
[@thenut77](http://twitter.com/)


Run Notes
-------------------------------------------------------------------------------
To run:
1) git clone
2) npm install
3) node still-pusher.js [options]
